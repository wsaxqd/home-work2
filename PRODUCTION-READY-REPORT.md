## 🚀 一键部署流程

### 步骤 1：准备 SSL 证书

```bash
# 方法 A：使用 Certbot（服务器上）
sudo certbot certonly --standalone -d qmzgai.com

# 方法 B：使用云厂商证书
# 从腾讯云/阿里云下载证书

# 复制证书到项目
cp fullchain.pem ssl/
cp privkey.pem ssl/
```

### 步骤 2：配置邮件服务

```bash
# 编辑 .env.production
nano .env.production

# 填写 SMTP 配置
SMTP_USER=your-email@qq.com
SMTP_PASSWORD=your-authorization-code
```

### 步骤 3：执行部署

```bash
# 一键部署
bash deploy-production.sh
```

就这么简单！🎉
