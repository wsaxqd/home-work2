## 🔍 检查网络问题

### 可能的原因:
1. **防火墙/网络限制:** 公司/学校网络可能屏蔽
2. **DNS问题:** 尝试更换DNS
3. **临时服务器问题:** 等待一段时间

### 尝试修复:
```bash
# Windows: 刷新DNS
ipconfig /flushdns

# 更换DNS为公共DNS:
# 设置 → 网络 → 更改适配器选项 → 右键网络 → 属性
# IPv4 → 属性 → 使用以下DNS:
# 首选: 223.5.5.5 (阿里DNS)
# 备用: 114.114.114.114
```
