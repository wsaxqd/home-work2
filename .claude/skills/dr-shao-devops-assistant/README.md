# Dr. 邵 DevOps Assistant

基于 Dr. 邵的 AI 辅助开发方法论的 Claude Code Skill。

## 快速开始

这个 skill 已经配置为全局可用。你可以直接开始使用：

```
→ "帮我配置 Nginx"
→ "连接腾讯云服务器"

```

## 核心方法：反问法

不知道怎么问时，先问 AI 需要什么信息！

### 示例对话

**你**: "我想部署到生产环境，但不知道从哪开始"

**AI**:
```
让我帮你梳理一下，我需要了解：
1. 你使用什么云服务商？（腾讯云/阿里云/AWS）
2. 你的应用是什么类型？（前后端分离/单体应用）
3. 你有域名了吗？
4. 数据库是自建还是云服务？

先告诉我这些，我会给你定制化的部署方案。
```

## 主要功能

### 1. 服务器管理
- SSH 连接配置
- 安全设置
- 文件操作指导

### 2. Nginx 配置
- 反向代理
- 负载均衡
- SSL/HTTPS
- 缓存和压缩
- 安全策略

### 3. Docker 部署
- docker-compose.yml 生成
- 开发/生产环境配置
- 容器管理
- 日志和监控

### 4. 自动化部署
- 部署脚本编写
- 无感知升级
- 版本管理
- 回滚策略

### 5. SSL 证书
- 免费申请（Let's Encrypt）
- 配置和挂载
- 自动续期

### 6. 问题排查
- 白屏/502 错误
- 容器启动失败
- SSL 不生效
- 性能优化
