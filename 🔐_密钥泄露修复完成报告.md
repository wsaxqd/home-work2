### Git 预提交钩子（建议）
```bash
#!/bin/bash
# .git/hooks/pre-commit

# 检查是否尝试提交 .env 文件
if git diff --cached --name-only | grep -E "\.env|.secrets|.credentials"; then
    echo "❌ 错误：不能提交敏感文件！"
    echo "请检查你的更改中是否包含："
    echo "  - .env 文件"
    echo "  - .secrets 文件"
    echo "  - 硬编码的密钥或密码"
    exit 1
fi
```

### 密钥轮换计划
- **JWT 密钥**：90 天轮换一次
- **数据库密码**：90 天轮换一次
- **第三方 API 密钥**：按服务商要求
- **会话 Secret**：每次大版本更新轮换
