# 腾讯云域名 DNS 快速配置 - qmzgai.com

## 🎯 5分钟快速上手

### 第一步：进入控制台（30秒）

1. 打开浏览器，访问：**https://console.cloud.tencent.com/domain**
2. 登录你的腾讯云账号
3. 在左侧菜单找到「我的域名」，点击进入

你会看到你购买的域名 `qmzgai.com`

---

### 第二步：进入解析设置（30秒）

在域名列表中找到 `qmzgai.com`，点击右侧的「解析」按钮

**或者：**
1. 点击域名进入详情页
2. 点击顶部的「解析」标签页

---

### 第三步：添加 DNS 记录（3分钟）

#### 📍 记录 1：主域名（qmzgai.com）

点击「添加记录」按钮，填写：

```
┌─────────────────────────────────────────┐
│ 主机记录：    @                          │
│ 记录类型：    A                          │
│ 线路类型：    默认                        │
│ 记录值：      [你的服务器IP]              │
│ TTL：        600（10分钟）                │
│ 权重：        留空                        │
└─────────────────────────────────────────┘
```

**示例：**
```
主机记录：@
记录类型：A
记录值：123.45.67.89
TTL：600
```

点击「保存」

---

#### 📍 记录 2：www 子域名（www.qmzgai.com）

再次点击「添加记录」：

```
┌─────────────────────────────────────────┐
│ 主机记录：    www                        │
│ 记录类型：    A                          │
│ 线路类型：    默认                        │
│ 记录值：      [你的服务器IP]              │
│ TTL：        600（10分钟）                │
└─────────────────────────────────────────┘
```

**示例：**
```
主机记录：www
记录类型：A
记录值：123.45.67.89
TTL：600
```

点击「保存」

---

#### 📍 记录 3：API 子域名（可选）

如果你的后端 API 需要独立域名：

```
┌─────────────────────────────────────────┐
│ 主机记录：    api                        │
│ 记录类型：    A                          │
│ 线路类型：    默认                        │
│ 记录值：      [你的服务器IP]              │
│ TTL：        600（10分钟）                │
└─────────────────────────────────────────┘
```

点击「保存」

## ⏱️ 生效时间说明

| 类型 | 生效时间 | 说明 |
|------|----------|------|
| 腾讯云 DNS | **即时生效** | 配置保存后立即生效 |
| 本地电脑 | 10-30 分钟 | 需要清除 DNS 缓存 |
| 全球 DNS | 最长 24 小时 | 受 TTL 值影响 |

**加速生效方法：**

```cmd
# Windows 清除 DNS 缓存
ipconfig /flushdns

# Mac 清除 DNS 缓存
sudo dscacheutil -flushcache

# Linux 清除 DNS 缓存
sudo systemd-resolve --flush-caches
```

## 🎨 进阶配置（可选）

### 1️⃣ 设置 301 跳转（www → 主域名）

如果你想让 `www.qmzgai.com` 自动跳转到 `qmzgai.com`：

**删除 www 的 A 记录**，添加 URL 转发：

```
主机记录：www
记录类型：显性URL
记录值：https://qmzgai.com
```

---

### 2️⃣ 开启智能解析（国内外分流）

如果你有国内外两台服务器，可以配置：

```
# 国内用户访问
主机记录：@
记录类型：A
线路类型：境内
记录值：[国内服务器IP]

# 海外用户访问
主机记录：@
记录类型：A
线路类型：境外
记录值：[海外服务器IP]
```

这样可以显著提升访问速度！

---

### 3️⃣ 添加邮箱 MX 记录（如需要）

如果要使用企业邮箱（如 admin@qmzgai.com）：

```
主机记录：@
记录类型：MX
记录值：mxbiz1.qq.com
优先级：5

主机记录：@
记录类型：MX
记录值：mxbiz2.qq.com
优先级：10
```

## 🛡️ 安全建议

### ✅ 必做：开启域名安全锁

1. 进入域名管理页面
2. 找到「域名安全」
3. 开启「**禁止转移锁**」
4. 开启「**禁止更新锁**」

**作用：** 防止域名被恶意转移或篡改

---

### ✅ 推荐：开启 DNSSEC

1. 在解析设置页面
2. 点击「DNSSEC 设置」
3. 点击「开启 DNSSEC」

**作用：** 防止 DNS 劫持和缓存投毒攻击

---

### ✅ 推荐：设置解析记录锁定

在解析记录列表中，点击每条记录右侧的「🔒」图标

**作用：** 防止 DNS 记录被误删或篡改

## ❌ 常见问题排查

### 问题 1：配置后无法访问网站

**可能原因：**
1. DNS 还未完全生效（等待 10-30 分钟）
2. 服务器防火墙未开放端口
3. Web 服务（Nginx/Apache）未启动

**解决方法：**
```bash
# 1. 检查服务器端口是否开放
telnet 你的服务器IP 80

# 2. 检查 Web 服务状态
sudo systemctl status nginx

# 3. 开放防火墙端口（如果使用防火墙）
sudo ufw allow 80/tcp
sudo ufw allow 443/tcp
```

---

### 问题 2：www 可以访问，主域名不行

**原因：** @ 记录配置错误或未配置

**解决方法：**
1. 检查 @ 记录是否存在
2. 确认记录值是否正确
3. 确认记录状态是「已启用」

---

### 问题 3：DNS 解析到错误的 IP

**原因：** 本地 DNS 缓存未更新

**解决方法：**
```cmd
# 清除本地 DNS 缓存
ipconfig /flushdns

# 重新验证
nslookup qmzgai.com
```

---

### 问题 4：国内访问慢或无法访问

**可能原因：**
1. 服务器在海外
2. 域名未备案（如果服务器在国内）

**解决方法：**
1. 使用腾讯云 CDN 加速
2. 如服务器在国内，进行 ICP 备案
3. 配置国内外分线路解析
