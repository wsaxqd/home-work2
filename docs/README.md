## 🚀 快速开始

### 1. 配置数据库
```bash
# 编辑 server/.env 文件
DB_HOST=localhost
DB_PORT=5432
DB_NAME=qmzg
DB_USER=your_user
DB_PASSWORD=your_password
```

### 2. 运行迁移
```bash
cd server
npm run migrate
```

### 3. 启动服务
```bash
npm run dev
```

### 4. 测试API
打开Postman，导入 `docs/api/1.1-register.md` 中的测试用例开始测试。
