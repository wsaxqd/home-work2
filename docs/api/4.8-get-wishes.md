# API 4.8 - 获取心愿墙

## 接口信息

- **接口名称**: 获取心愿墙
- **请求方法**: GET
- **请求路径**: `/api/community/wishes`
- **需要认证**: 否

---

## 接口描述

获取心愿墙上的所有心愿列表，支持分页和筛选。

---

## 请求参数

### Query参数
| 参数名 | 类型 | 必填 | 说明 | 默认值 |
|--------|------|------|------|--------|
| page | Integer | 否 | 页码 | 1 |
| pageSize | Integer | 否 | 每页数量 | 20 |
| category | String | 否 | 心愿分类 | 全部 |
| status | String | 否 | 心愿状态(pending/achieved) | 全部 |
| sortBy | String | 否 | 排序方式(time/support) | time |

### 请求示例

```
GET /api/community/wishes?page=1&pageSize=20&category=学习&sortBy=support
```

## 字段说明

### 响应字段
| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | UUID | 心愿ID |
| content | String | 心愿内容 |
| category | String | 心愿分类 |
| status | String | 心愿状态 |
| support_count | Integer | 支持数 |
| created_at | DateTime | 发布时间 |
| achieved_at | DateTime | 实现时间（未实现为null） |
| user | Object | 发布者信息 |

---

## 业务规则

1. 默认按时间倒序排列
2. 支持按支持数排序
3. 可以按分类筛选
4. 可以按状态筛选（进行中/已实现）
5. 已删除的心愿不返回

---

## 使用场景

- 浏览心愿墙
- 查看他人心愿
- 为心愿加油
- 寻找灵感

---

## 注意事项

1. 建议使用分页加载
2. 可以缓存热门心愿
3. 心愿内容应支持换行显示
4. 已实现的心愿可以特殊标识
