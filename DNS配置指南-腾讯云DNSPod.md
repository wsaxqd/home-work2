# 腾讯云 DNSPod 配置指南 - qmzgai.com

## 📌 前置要求

- 已在腾讯云购买域名或域名已转入 DNSPod
- 已准备好服务器公网 IP 地址
- 域名实名认证已完成

---

## 🔧 配置步骤

### 第一步：登录 DNSPod 控制台

1. 访问：https://console.dnspod.cn/
2. 登录腾讯云账号
3. 点击「我的域名」

### 第二步：选择域名

1. 在域名列表中找到 `qmzgai.com`
2. 点击域名进入解析设置页面

### 第三步：添加主域名 A 记录

点击「添加记录」：

| 字段 | 值 | 说明 |
|------|-----|------|
| **主机记录** | `@` | 代表主域名 qmzgai.com |
| **记录类型** | `A` | IPv4 地址 |
| **线路类型** | `默认` | 所有用户 |
| **记录值** | `你的服务器IP` | 例如：123.45.67.89 |
| **TTL** | `600` | 10分钟 |
| **MX优先级** | 留空 | A记录不需要 |

点击「保存」。

### 第四步：添加 www 子域名

再次点击「添加记录」：

| 字段 | 值 | 说明 |
|------|-----|------|
| **主机记录** | `www` | www.qmzgai.com |
| **记录类型** | `A` | IPv4 地址 |
| **线路类型** | `默认` | 所有用户 |
| **记录值** | `你的服务器IP` | 与主域名相同 |
| **TTL** | `600` | 10分钟 |

### 第五步：添加 API 子域名（可选）

| 字段 | 值 | 说明 |
|------|-----|------|
| **主机记录** | `api` | api.qmzgai.com |
| **记录类型** | `A` | IPv4 地址 |
| **线路类型** | `默认` | 所有用户 |
| **记录值** | `你的服务器IP` | 与主域名相同 |
| **TTL** | `600` | 10分钟 |

## ✅ DNSPod 特色功能

### 1. 智能线路解析（推荐）

DNSPod 支持国内外分线路解析，提升访问速度：

**配置示例：**

| 主机记录 | 类型 | 线路类型 | 记录值 | 说明 |
|---------|------|---------|--------|------|
| @ | A | 默认 | 国外服务器IP | 兜底线路 |
| @ | A | 境内 | 国内服务器IP | 国内用户访问 |
| @ | A | 境外 | 国外服务器IP | 海外用户访问 |

### 2. 负载均衡

如果有多台服务器：

```
主机记录：@
记录类型：A
记录值：123.45.67.89（权重：100）
记录值：123.45.67.90（权重：100）
```

DNSPod 会自动轮询分配流量。

### 3. URL 转发（301/302）

如果需要将 www 强制跳转到主域名：

```
主机记录：www
记录类型：显性URL
记录值：https://qmzgai.com
```

## 🔍 验证 DNS 配置

使用 DNSPod 提供的诊断工具：

1. 在域名列表页面点击「诊断工具」
2. 输入 `qmzgai.com`
3. 查看解析结果

或使用命令行：
```bash
nslookup qmzgai.com 119.29.29.29
```

（119.29.29.29 是 DNSPod 的公共 DNS）

## 🛡️ 安全设置

### 1. 开启域名锁定

防止域名被恶意转移：
1. 进入域名管理页面
2. 找到「域名安全」
3. 开启「域名锁定」

### 2. 开启解析记录锁定

防止 DNS 记录被篡改：
1. 在解析记录列表
2. 点击记录右侧「锁定」图标

### 3. 设置 API Token 权限

如果使用 API 操作：
1. 进入「用户中心」→「安全设置」
2. 创建 API Token
3. 限制 IP 白名单
