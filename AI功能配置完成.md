## 🔄 其他AI服务选项

### 如果想使用DeepSeek (性价比更高):

1. **注册DeepSeek账号**
   - 访问: https://platform.deepseek.com/
   - 注册并获取API密钥

2. **配置密钥**
   - 打开 `server\.env`
   - 修改:
     ```
     AI_PROVIDER=deepseek
     DEEPSEEK_API_KEY=sk-你的密钥
     ```

3. **重启服务器**

### 如果想使用智谱AI GLM:

1. **注册智谱AI账号**
   - 访问: https://open.bigmodel.cn/
   - 注册并获取API密钥

2. **配置密钥**
   - 打开 `server\.env`
   - 修改:
     ```
     AI_PROVIDER=zhipu
     ZHIPU_API_KEY=你的密钥
     ```

3. **重启服务器**
