# Dify AI 配置步骤 - 跟我一起做

## 🎯 第一步: 创建AI对话助手应用

### 1. 登录Dify控制台

你现在应该已经登录到: https://cloud.dify.ai

看到的页面应该是这样的:
```
┌─────────────────────────────────────┐
│  Dify                               │
│  [+ Create New App]  [My Apps]      │
├─────────────────────────────────────┤
│                                     │
│  你的应用列表会显示在这里            │
│                                     │
└─────────────────────────────────────┘
```

---

### 2. 点击"Create New App"按钮

找到并点击蓝色的 **"+ Create New App"** 按钮

---

### 3. 选择应用类型

会弹出一个对话框,显示几种应用类型:

**请选择:** **"Chatbot"** 或 **"Chat App"** (聊天机器人)

图标可能是一个对话气泡 💬

---

### 4. 选择创建方式

你会看到两个选项:

- **"Create from Blank"** (从空白开始) ← **选这个!**
- "Create from Template" (从模板创建)

**点击 "Create from Blank"**

---

### 5. 填写应用基本信息

现在会出现一个表单,请填写:

**Application Name (应用名称):**
```
启蒙之光-AI助手
```

**Description (描述,可选):**
```
儿童AI学习助手,帮助6-12岁儿童学习和成长
```

**Icon (图标,可选):**
- 可以选择一个可爱的emoji
- 或者上传自定义图标
- 建议选: 🤖 或 📚 或 ✨

填写完成后,点击 **"Create"** 按钮

---

### 6. 进入应用配置页面

创建成功后,你会进入应用的配置页面。

页面左侧是菜单:
```
- Studio (工作室)
- Orchestrate (编排)
- Prompt (提示词)
- Model (模型)
- Features (功能)
- API Access (API访问)
```

---

### 7. 配置AI模型

**点击左侧菜单的 "Orchestrate" 或 "编排"**

在右侧配置区域,你会看到:

#### 模型设置:

**Model Provider (模型供应商):**
- 选择: **OpenAI** (如果你有OpenAI API Key)
- 或选择: **Anthropic Claude** (推荐,效果好)
- 或选择: 其他你已配置的模型

**Model (模型):**
- GPT-3.5-turbo (便宜,快速) ← **推荐开始用这个**
- GPT-4 (更聪明,更贵)
- Claude-3 (Anthropic,效果好)

**Temperature (温度):**
```
0.7
```
(控制创意程度,0.7比较平衡)

**Max Tokens (最大输出长度):**
```
2000
```

---

### 8. 配置系统提示词(非常重要!)

找到 **"System Prompt"** 或 **"系统提示词"** 输入框

**复制下面的内容粘贴进去:**

```
你是"启启",一个专为6-12岁儿童设计的AI学习助手。

你的特点:
- 用简单易懂的语言与孩子交流
- 富有耐心和鼓励性
- 善于用生动的例子解释概念
- 关注孩子的情感需求
- 引导孩子独立思考,而不是直接给答案

你的任务:
- 回答孩子的各种问题
- 帮助解释学习中的困惑
- 提供学习建议和方法
- 给予鼓励和表扬
- 培养良好的学习习惯

交流规则:
- 使用简单的词汇,避免复杂术语
- 每次回答不超过150字
- 可以适当使用emoji让对话更生动😊
- 保持积极正面的态度
- 如果孩子问到不适合的内容,温和地引导到正面话题

记住:你是孩子的朋友和导师,要让他们感到被理解和支持。
```

---

### 9. 测试应用

在配置页面右侧,应该有一个 **"Preview"** 或 **"预览"** 面板

试着输入:
```
你好启启,你能帮我学习吗?
```

看看启启是否会回复,如果回复正常,说明配置成功! ✅

---

### 10. 发布应用

配置完成后,点击右上角的 **"Publish"** 或 **"发布"** 按钮

---

### 11. 获取API Key (重要!)

**点击左侧菜单的 "API Access" 或 "API访问"**

你会看到一个 **API Key**,格式类似:
```
app-xxxxxxxxxxxxxxxxxxxx
```

**请复制这个API Key,发给我!**

这是第一个应用的Key,我们还需要创建另外两个应用。
